# Extract pairs of regex capture groups from all matches in `text`. `pattern` must have exactly two capturing groups. Returns a list of `character(2)` vectors, one per match.

Extract pairs of regex capture groups from all matches in `text`.
`pattern` must have exactly two capturing groups. Returns a list of
`character(2)` vectors, one per match.

## Usage

``` r
.regex_all_captures2(text, pattern)
```
